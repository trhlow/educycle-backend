# EduCycle Backend

EduCycle is a peer-to-peer academic book exchange platform for students.
This repository contains the backend REST API.

## Core Features
- User authentication (JWT)
- Book product management by category
- Transaction lifecycle with OTP confirmation
- In-app chat per transaction
- Rating and review system

## Out of Scope
- Online payment
- Dispute resolution
- Admin panel
- AI / recommendation system

## Tech Stack
- ASP.NET Web API (.NET 8)
- Entity Framework Core
- SQL Server
- Swagger / OpenAPI

## Development Rules
- No issue → no code
- No direct commit to main
- Feature-based branching
